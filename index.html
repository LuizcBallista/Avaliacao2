Documentação SWAPI

Este site tem como objetivo apresentar uma breve documentação prática sobre a utilização da API 'swapi.dev', que contém vários dados referentes ao universo fictício de Star Wars, seus personagens, planetas e filmes.

Método 1 - Filmes

O primeiro método utilizado é o "films", que retorna todos os filmes clássicos de Star Wars lançados até 2005 (A vingança dos Sith). Para se comunicar com este método da API é necessário acrescentar o endpoint films à url original, 'https://swapi.dev/api/films'.

Neste caso, está sendo utilizado uma varredura de array para mostrar os títulos dos filmes.

Ao clicar no botão "Buscar Filmes" trazemos os filmes da API e os inserimos dentro uma nova tag "p".

Método 2 - Pessoas

O segundo método utilizado é o 'people' e será responsável por trazer as pessoas, ou seja, os personagens dos filmes de Star Wars. Para se comunicar com este método da API é necessário acrescentar o endpoint people à url original'https://swapi.dev/api/people'.

Os personagens de Star Wars são bem conhecidos em todo o mundo, até mesmo para não fãs! Claro, há muitos personagens na franquia.

Este método busca os personagens na API através do ID informado no campo abaixo.

Método 3 - Planetas

A demonstração do terceiro método é mais complexa que os anteriores. A primeira parte dele funciona exatamente igual ao segundo método, é informado um ID juntamente com um endpoint fixo que realiza a consulta na API, 'https://swapi.dev/api/planets'. Porém, depois de recuperadas as informações do planeta, é feito uma verificação se existem residentes nele, caso existam, é retirado o ID de cada residente e passado para o método de 'buscarPessoas'

Ou seja, é um método que chama outro método, retornando dois (ou mais) resultados por consulta. Foram necessárias várias validações para permitir o funcionamento correto.

As informações do planeta são fixas, sendo trazidas de forma estática. Já os residentes são dinamicamente gerados através da quebra da URL pelo método split(), para retirarmos o ID do residente.

Este método busca as pessoas que vivem no planeta e mostra todas em uma nova Tag "p"
